# 向心链 - Excel 数据处理工具

一个功能强大的在线工具，提供 Excel 转 HTML 和 Excel 合并两大核心功能，让数据处理更加高效便捷。

## 核心功能

### 1. Excel 转 HTML

将 Excel 文件转换为美观、响应式的 HTML 表格，支持以下特性：

- **支持多种格式**：兼容 `.xlsx` 和 `.xls` 格式
- **智能处理**：自动识别链接和图片列，转换为可点击链接
- **响应式设计**：在桌面端显示表格视图，在移动端自动切换为卡片视图
- **内置搜索**：支持多关键词搜索，快速定位数据
- **列管理**：自动隐藏指定列（拍卖平台、标的类型、省份、城市）
- **拖拽上传**：支持文件拖拽上传，操作更便捷
- **自定义保存**：可配置文件名、编码格式和样式选项
- **纯前端实现**：无需后端服务，数据处理在本地完成

### 2. Excel 合并

将多个 Excel 文件合并为一个，提供以下功能：

- **多文件支持**：同时选择多个 Excel 文件进行合并
- **表结构验证**：自动验证文件结构是否一致，确保合并质量
- **智能去重**：可配置去重依据列，支持全列匹配去重
- **数据清理**：自动跳过空行，保持数据整洁
- **格式保留**：尽可能保留原始数据的格式和样式
- **数据预览**：合并后显示前 100 行数据，支持搜索和分页
- **详细统计**：显示总行数、合并文件数、处理时间和去重行数
- **操作日志**：记录处理过程，方便排查问题

## 使用方法

### Excel 转 HTML

1. 打开 `index.html` 页面
2. 上传 Excel 文件（支持拖拽）
3. 点击"转换为 HTML"按钮
4. 点击"另存为 html"下载生成的 HTML 文件

### Excel 合并

1. 打开 `index.html` 页面并切换到"Excel 合并"标签
2. 选择多个 Excel 文件（支持拖拽）
3. 点击"验证表结构"确认文件结构一致性
4. 配置合并选项（去重、格式保留等）
5. 点击"开始合并"执行合并操作
6. 预览合并结果后点击"保存"下载合并后的文件
